# RagChunk 一期说明

## 离线建库流程（5 步）

1. 创建知识库：通过 API 传递切片、索引、检索等参数。
2. 上传文件：支持 TXT、Markdown、DOCX。
3. 解析与文本规范化：统一换行、压缩空白。
4. 混合切片：先规则切片，再按质量分决定是否调用千问语义重切。
5. 向量化入库：使用 Embedding 写入本地向量库。

## 在线问答

用户提问后，系统对问题向量化，在知识库中检索 TopK 条分段，再交给大模型生成回答。

## 千问切片触发

- aiMode 为 never 时不调用千问。
- aiMode 为 auto 时，质量分低于 70 或长文仅一段时会触发。
- 上传参数 smartChunk=true 会强制触发语义重切。
